如何优雅地用Rmarkdown发布Hexo文章？

June 11, 2019 R 本文总阅读量次

Markdown是一个基于Web的文本到HTML的转换工具，可以通过简单、以读写的文本格式生成结构化的HTML文档。
为什么选择Rstudio作为markdown的编辑环境？
答案参考知乎《RStudio的markdown功能怎么用起来，和其他markdown软件的异同？》
https://www.zhihu.com/question/35803490

准备工作：
install.packages(c(‘ggplot2’, ‘knitr’, ‘foreign’, ‘rmarkdown’, ‘stargazer’,”htmltools”,”caTools”,”bitops” ))

1.打开Rstudio → Tools → Global Options
2.点击sweave，在Weave Rnw files using选择knitr; Typeset Latex into PDF using 选择XeLaTex.
3.File → New file → R Markdown.
4.输入Title和Author，选择Html，然后OK.
5.接下来就可以在新窗口中优雅地写文章啦.
6.文章写好后，在Knit中选择不同的文件类型进行预览，并保存在桌面.
7.将生成.Rmd文件复制到你的hexo文件夹中source下的_posts 中.
8.执行hexo命令:hexo g(用于生成静态文件);hexos(用于启动服务器，进行本地预览)；完成后打开浏览器输入localhost:4000查看本地发布效果；hexo deploy(发布文章到github.io).
9.大功告成！
